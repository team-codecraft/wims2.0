# WIMS 2.0 프로젝트 일정표

> **사업명:** WIMS 시스템 개선 개발
> **개발기간:** 2026.03.23 ~ 2026.11.01 (8개월, 16 스프린트)
> **방법론:** 특허청 SW 개발방법론 (객체지향 CBD 기반)
> **문서버전:** v1.1
> **최종 수정일:** 2026.04.27

---

## 1. 전체 일정 개요

프로젝트는 2개의 Phase로 나뉘어 진행되며, 총 16개 스프린트(S0~S15)와 5개의 Gate로 구성된다.

- **Phase 1 (M1~M3, 2026.03.23 ~ 2026.06.14):** BOM 제품관리 시스템 + MES REST API 연동
- **Phase 2 (M4~M8, 2026.06.15 ~ 2026.11.01):** 견적설계, 발주관리, 제조관리, 현장실측 시스템

## 4. Gate 기준 및 진입 조건

### Gate 1 — 분석 완료 (S1 종료, M1 말)

- 요구사항 정의서 사업주 승인
- MES REST API 인터페이스 규격 합의 (MES 팀 서명)
- As-Is/To-Be 업무흐름도 확정 (5개 시스템)
- 총괄 테스트 계획서 작성 완료

### Gate 2 — 설계 완료 (S2 종료, M2 1주차)

- BOM 논리/물리 ERD 확정
- MES REST API 규격서 확정 및 MES 팀 서명
- 화면 설계서 확정 (앱 UI 포함)
- 데이터 전환 매핑 정의 완료


### ★ Gate 3 — Phase 1 오픈 (S5 종료, M3 말) ※ 사업주 참여 필수

- Phase 1 통합 테스트 100% 통과
- MES REST API 연동 운영 검증 완료 (MES 팀 서명)
- Critical 결함 0건

### Gate 4 — Phase 2 구현 완료 (S11 종료, M6 말)

- 전 스프린트(S6~S11) 단위 테스트 통과
- 핵심 로직 코드 커버리지 80% 이상
- 현장실측 앱 파일럿 완료
- Major 이상 결함 0건

### ★ Gate 5 — Phase 2 전체 오픈 (S15 종료, M8) ※ 사업주 참여 필수

- UAT 사업주(유미숙) 서명 완료 (웹 + 앱)
- **초기 데이터 시드 입력 검증 완료** (자재 마스터·사용자·코드값)
- Critical/Major 결함 0건
- 운영 매뉴얼 인도 완료 (5개 시스템 + 앱)

**Gate 운영 원칙:** Gate 미통과 시 PP11-1 §7 에스컬레이션 절차에 따라 PM→사업주 보고 후 보완 계획을 수립한다. Gate 3/5는 사업주(유니크시스템 유미숙) 참여 필수이며, 검토 결과는 회의록으로 기록하여 형상관리 저장소에 보관한다. 변경 발생 시 PP11-1 §8 변경관리 절차(CR 등록→영향도 분석→승인)를 적용한다.
