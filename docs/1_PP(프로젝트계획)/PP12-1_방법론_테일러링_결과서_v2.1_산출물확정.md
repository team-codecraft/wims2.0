# **4. 단계별 Gate 기준**

각 단계 완료 시 게이트(Gate) 검토를 실시합니다. Gate 3(Phase 1 오픈)과 Gate 5(Phase 2 오픈)은 핵심 게이트(★)로서 사업주(유미숙) 참여 검토를 의무화합니다.

| **Gate** | **시점** | **진입 기준** |
