### 1.1 제품 구조 (창호 구성)

| 항목 | 값 |
|---|---|
| **구성** | **가로 2 × 세로 2 = 4짝 미서기 이중창** |
| 전체 가로길이 변수 | **W** |
| 전체 높이 변수 | **H** |
| 보조 표기 (시트 헤더) | **W1 = W/4, H1 = H/2** |

→ 한 짝(Vent) 1개의 가로폭 ≈ `W/4`, 높이 ≈ `H/2` (외/내 + 좌/우 4분할 기준)
→ 길이 단위(M) 자재의 Q'TY 컬럼은 **공란이 아니라 "규격" 컬럼의 공식으로 길이를 산정**하는 방식 (예: 모헤어 H/2 = 위치당 H/2 미터)

### 1.2 기타 메타정보

|---|---|---|
