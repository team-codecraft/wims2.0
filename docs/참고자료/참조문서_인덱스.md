# WIMS 2.0 참조 문서 인덱스

Vault 전체의 얇은 진입 허브. 단계별 상세 문서는 **하위 인덱스**로 진입하고, 그래프뷰에서 이 노트를 중심 허브로 확장한다. 스프린트 현황은 [[STATUS]].

> [!info] 사용 지침
> - 특정 단계 문서를 찾을 때는 **하위 인덱스 먼저** 진입
