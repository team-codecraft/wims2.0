## 1. 개요

### 파일 목적

유니크시스템이 제조하는 **미서기창호**와 **커튼월** 전 제품 라인업을 계약구분(마스제품/우수제품), 유리사양, 프레임 치수, 제품 유형(기본/파생)에 따라 체계적으로 정리한 제품 카탈로그 테이블.

### 시트 구성

| 시트명 | 행 수 | 열 수 | 설명 |
|--------|-------|-------|------|
| 커튼월 및 미서기 분류 | 35행 | 23열 | 단일 시트에 미서기(상단)·커튼월(하단) 두 제품군을 계층적으로 배치 |

- **삽입 이미지 없음** (미디어 파일 미포함)
- 셀 병합을 적극 활용한 계층 헤더 구조

## 3. 제품 코드 체계 / 네이밍 규칙

### 3-1. 미서기 모델 코드 구조

#### 마스제품 코드 패턴

```
{브랜드접두사}-{제품유형코드}{치수}-{연결유형}-{버전}
```

| 세그먼트 | 값 예시 | 의미 |
|----------|---------|------|
| 브랜드접두사 | `DHS` | 구(舊) 디에이치에스 브랜드 (구형 마스제품) |
| 브랜드접두사 | `UNI` | 유니크 브랜드 |
| 제품유형코드 | `AE` | 단열(Airtight/단열 알루미늄 미서기 추정) |
| 제품유형코드 | `WA` | 미서기 변형 라인 (WA계열) |
| 치수 | `225`, `226`, `160` | 프레임 기준치수 (mm) |
| 연결유형 | `D` | 중중연 (Double + 중간레일 2개) |
| 연결유형 | `B` | 중연 (Double + 중간레일 1개) |
| 연결유형 | `A` | 단순 이중창 |
| 버전/파생 | `1` | 기본제품 버전1 |
| 파생 | `S` | 파생(Subsidiary) — 1mm 축소 치수 버전 |

**예시 해독**
- `DHS-AE225-D-1`: DHS 브랜드 / AE계열 225mm / 중중연(D) / 기본버전1
- `UNI-AE224-D-S`: UNI 브랜드 / AE계열 224mm / 중중연(D) / 파생(S)
- `UNI-AE160`: UNI 브랜드 / AE계열 160mm / 연결유형·버전 생략(단창 단순구조)

#### 우수제품 코드 패턴 (미서기)

```
U-P-{제품유형코드}{치수}-{연결유형}-{파생}
```

| 세그먼트 | 값 | 의미 |
|----------|----|------|
| `U` | 고정 | 유니크(Unique) |
| `P` | 고정 | 우수제품(Premium/우수 인증) |
| `SM` | 제품유형 | Sliding(미서기) 추정 |
| 치수 | `225`, `160` | 프레임 기준치수 (mm) |
| 연결유형 | `D`, `B`, `A` | 마스제품과 동일 |
| 파생 | `S` | 파생(1mm 축소) |
| 파생 | `B` | 중연 변형 (SM160-B 사례: 별도 명칭 부여) |

**예시 해독**
- `U-P-SM225-D`: 우수제품 / 미서기 225mm / 중중연
- `U-P-SM224-D-S`: 우수제품 / 미서기 224mm / 중중연 / 파생

### 3-2. 커튼월 모델 코드 구조

#### 마스제품 코드 패턴

```
{브랜드}-{제품유형}{치수}{히든여부}-{유리레이어수}{파생구분}
```

| 세그먼트 | 값 | 의미 |
|----------|----|------|
| 브랜드 | `DHS` / `UNI` | DHS=구형, UNI=파생 |
| 제품유형 | `CW` | Curtain Wall |
| 치수 | `206`, `176` 등 | 프레임 기준치수 (mm) |
| 히든여부 | `H` 접미사 | 히든바(Hidden bar) 방식 |
| 유리레이어 | `3` | 삼중유리 |
| 유리레이어 | `2` | 복층유리 |
| 파생구분 | `S` | 파생1 (반강화유리) |
| 파생구분 | `SS` | 파생3 (43mm / 소방창) |

**예시 해독**
- `DHS-CW206-3`: DHS / 커튼월 206mm / 캡바 / 삼중유리 / 기본
- `DHS-CW180H-3`: DHS / 커튼월 180mm / **히든바** / 삼중유리 / 기본
- `UNI-CW179H-3S`: UNI / 커튼월 179mm / 히든바 / 삼중유리 / **파생(반강화)**

#### 우수제품 코드 패턴 (커튼월)

```
U-P-{CW}{치수}{히든여부}-{유리레이어수}{파생구분}
```

- `U-P-CW207-3`: 우수제품 / 커튼월 207mm / 삼중유리 / 기본
- `U-P-CW181H-3`: 우수제품 / 커튼월 181mm / 히든바 / 삼중유리
- `U-P-CW208-3S`: 우수제품 / 커튼월 208mm / 삼중유리 / 파생1(반강화)
- `U-P-CW204-3SS`: 우수제품 / 커튼월 204mm / 삼중유리 / 파생3(43mm 소방창)
- `U-P-CW189-24`: 우수제품 / 커튼월 189mm / 복층유리(숫자2) / 24년형(24 접미사)

### 3-3. 제작도·GMB 명칭 패턴

```
{연도}년-{치수}AL-{계약구분}{버전}
```

- `19년-225AL-마스1`: 2019년 설계 / 225mm 알루미늄 / 마스제품1호
- `24년-160AL-우수1`: 2024년 설계 / 160mm 알루미늄 / 우수제품1호
- 커튼월: `마스-{너비}*60*두께T`, `우수-{너비}*60*두께T` 형식 (가로*세로*두께)

## 5. WIMS PM 서브시스템 관점의 시사점

### 5-1. 제품 분류 축 (카탈로그 계층)

제품 카탈로그는 다음 4개 축의 조합으로 구성됨:

```
대분류(제품군) > 계약구분 > 유리사양 > 치수/크기구분
     ↓               ↓          ↓           ↓
  미서기/커튼월   마스/우수    단창/이중창   160/225/206/...
```

PM 서브시스템의 `제품분류(ProductCategory)` 엔티티는 **최소 4단계 계층**을 지원해야 한다.

### 5-2. 기본제품 vs 파생제품 관계

- 기본제품과 파생제품은 **치수가 1mm 차이**나는 쌍(pair) 관계임
  - 예: `DHS-AE225-D-1`(기본, 225mm) ↔ `UNI-AE224-D-S`(파생, 224mm)
- 커튼월은 **최대 6개 파생**(캡바/히든바 × 기본/반강화/43mm)이 존재
- WIMS EBOM(자재구성)에서 기본제품 BOM을 마스터로 두고, 파생제품은 **변형 BOM(Variant BOM)** 으로 관리하는 구조 권장

### 5-3. 모델 코드 속성으로 분해 가능한 항목

PM 제품 속성(ProductAttribute) 설계 시 모델 코드에서 자동 파싱 가능한 필드:

| 속성명 | 파싱 방법 | 예시 |
|--------|-----------|------|
| 브랜드 | 코드 첫 세그먼트 | `DHS`, `UNI`, `U-P` |
| 제품유형 | 두 번째 세그먼트 타입부 | `AE`, `WA`, `SM`, `CW` |
| 프레임치수(mm) | 숫자 추출 | `225`, `180`, `206` |
| 히든바 여부 | `H` 접미사 존재 여부 | boolean |
| 유리사양 | `2`(복층)/`3`(삼중) | enum |
| 파생구분 | 말미 `S`, `SS` 여부 | 기본/파생1/파생2 |
| 계약구분 | `U-P` 접두사 여부 | 마스/우수 |
| 연결유형(미서기) | `A`/`B`/`D` 세그먼트 | 단/중연/중중연 |

### 5-4. BOM 매핑 힌트

- **마스 제품** = `DHS-*` / `UNI-*` 코드 → 기존 BOM 문서(`DHS-AE225-D-1_BOM_정리.md`)와 직접 연결 가능
- **우수 제품** = `U-P-*` 코드 → 신규 BOM 설계 필요 (현재 참고자료 미확보)
- `제작도·GMB 명칭`의 **연도 정보**는 BOM 버전 관리(개정 이력)에 활용 가능
- 커튼월의 `너비*60*두께T` 패턴은 **단면 규격(단면코드)**으로, 부재(Profile) BOM의 `단면코드` 속성으로 매핑 권장

### 5-5. 확인 필요 사항

1. **커튼월 삼중유리 특대(236MM)** — 마스/우수 모두 데이터 없음. 미개발 예정 제품인지 확인 필요
2. **226MM 계열 미서기(WA226)** — 파생제품 없음. WA 시리즈의 단종/현행 여부 확인
3. **우수제품 커튼월 복층유리** — 제작도 명칭 일부 누락(`U-P-CW193H-24` 등). 원본 재확인 필요
4. **우수제품 커튼월 히든바 제품명** — `U-P-CW181H-3`, `U-P-CW151H-3` 제품명 셀 = `단열히든커튼월`만 기재 (치수 누락). 원본 재확인 필요
