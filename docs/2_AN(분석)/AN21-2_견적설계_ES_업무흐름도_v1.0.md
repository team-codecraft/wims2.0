## 1. As-Is 현행 업무 프로세스

### 1.1 개요

현행 견적설계는 WIMS에서 가장 많은 사용자(6명 중 5명)가 사용하는 핵심 업무 영역이다. CAD 도면 기반 견적 설계를 수행하나, 롤방충망 자동산출 부재(전원 요청), CAD 블록 수동 인식, 수동 보강 적용 등 자동화 미흡이 주요 불만이다. 만족도 3.0점(보통).

### 1.2 현행 업무 흐름도

```mermaid
flowchart TD
    Start([견적설계 시작])

    Start --> A1[프로젝트 생성<br/>유형/이름/주소/일정/고객정보]
    A1 --> A2[설계자 배정<br/>담당자 수동 선택]

    A2 --> B1[CAD 도면 업로드<br/>.dwg/.dxf 파일]
    B1 --> B2[도면 뷰어에서 확인<br/>Canvas 기반 렌더링]
    B2 --> B3[개소 수동 라벨링<br/>A-1, B-2 등 마킹]
    B3 --> B4{CAD 블록<br/>인식?}
    B4 -->|수동 식별| B5[블록/외부참조 수동 확인<br/>⚠️ 시간 소요 큼]

    B5 --> C1[제품 매핑<br/>개소별 창호 제품 선택]
    C1 --> C2[BOM 수동 입력<br/>자재/수량 직접 기입]
    C2 --> C3{롤방충망<br/>산출?}
    C3 -->|수동| C4[Excel에서 별도 계산<br/>⚠️ 전원 불편 호소]
    C3 -->|미적용| C5[방충망 제외]

    C4 --> D1[견적 계산 수행<br/>자재비+시공비+별도비]
    C5 --> D1
    D1 --> D2[실행예산 작성<br/>제작비/시공비/수수료/별도비]
    D2 --> D3[실행율 수동 계산<br/>⚠️ 정확도 이슈]

    D3 --> E1[견적서 생성]
    E1 --> E2[견적서 수동 편집<br/>⚠️ 메모 복사 불편]
    E2 --> E3{복수 개체<br/>변경?}
    E3 -->|개별 수정| E4[하나씩 수동 변경<br/>⚠️ 대량 작업 시 비효율]

    E4 --> F1[견적서 확정/출력]
    F1 --> End([종료])
```

## 2. To-Be 목표 업무 프로세스

### 2.1 개요

WIMS 2.0 견적설계는 CAD 블록 자동 인식, 롤방충망 자동산출, 미서기 보강 자동화, 구조검토 자동화를 통해 수작업을 최소화한다. 복수 개체 일괄 변경, 견적서 수정 편의성 개선으로 대량 프로젝트 효율을 높인다.

### 2.2 목표 업무 흐름도

```mermaid
flowchart TD
    Start([견적설계 시작])

    %% ── A. 프로젝트 설정 ──
    Start --> A1[프로젝트 선택/등록<br/>PM 시스템 연계]
    A1 --> A2[설계자 배정 + 진행현황 관리<br/>FR-ES-003]

    %% ── B. 도면 분석 ──
    A2 --> B1[CAD 도면 업로드 + 웹 뷰어 변환<br/>FR-ES-001 · NFR-PF-ES-001]
    B1 --> B2[CAD 블록/외부참조 자동 인식<br/>FR-ES-004]
    B2 --> B3[개소 자동 라벨링 + 수동 보정<br/>FR-ES-002]
    B3 --> B4[창호부호/수량 자동 입력<br/>FR-ES-011]

    %% ── C. 제품 설계 및 BOM 구성 ──
    B4 --> C1[개소별 제품 매핑<br/>창호·교실출입문·갤러리도어 등<br/>FR-ES-005 · FR-ES-012]
    C1 --> C2[개소별 치수 입력<br/>FR-ES-006]
    C2 --> C3[제품-BOM 연동 자재 구성 자동 반영<br/>롤방충망 자동산출 포함<br/>FR-ES-007 · FR-ES-008]
    C3 --> C4[미서기 보강 자동 적용<br/>FR-ES-009]
    C4 --> C5[구조검토 자동 수행 + 보강 반영<br/>FR-ES-010]
    C5 --> C6[트랜섬 절단 사이즈 자동계산<br/>FR-ES-013]

    %% ── D. 견적 산출 ──
    C6 --> D1[자재비·시공비·기타비용 합산<br/>견적서 자동 산출<br/>FR-ES-016]
    D1 --> D2[실행예산 항목별 비용 산출 + 수동 조정<br/>FR-ES-018]
    D2 --> D3[실행율 자동 연산<br/>FR-ES-017]
    D3 --> D4[실행예산 총괄 분석<br/>원가율·수익성 검토<br/>FR-ES-020]

    %% ── E. 견적서 검증 및 편집 ──
    D4 --> E1[식별번호 매칭 부호 중량 검증<br/>FR-ES-021]
    E1 --> E2{검토·수정 필요?}
    E2 -->|Yes| E3[견적서 수정<br/>복수 개체 일괄변경 · 메모 복사<br/>FR-ES-014 · FR-ES-015]
    E3 --> E1
    E2 -->|No| E4[견적서 확정]

    %% ── F. 출력 및 현장실측 피드백 ──
    E4 --> F1[견적서 출력<br/>발주처 제출용 PDF<br/>FR-ES-019]
    F1 --> F2{현장실측 데이터<br/>수신?}
    F2 -->|수신| F3[실측 데이터 반영<br/>FS 시스템 연계 FR-FS-005]
    F3 --> C2
    F2 -->|미수신| End([견적설계 완료])
```
