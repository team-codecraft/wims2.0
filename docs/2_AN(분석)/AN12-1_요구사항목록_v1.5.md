### 4.1 사이트맵 설명 미기재 화면 (확인 필요 14건)

사이트맵 코멘트 파일에서 설명이 없는(NaN) 화면 중 요구사항 매핑이 불명확한 항목이다. 현업 확인이 필요하다.

| 화면 | 경로 | 이슈 |
|------|------|------ |
| 순번 목록/상세 | /pages/product/sequence | 기능 불명 — 요구사항 미매핑 |
| 견적설계 설정/계산/실행 탭 | /config, /calculation, /execution | 사이트맵 설명 없음 — FR-ES 매핑은 추정 |
| 주문서 | /orderForm (견적설계 하위) | 발주 탭과 동일 경로인지 확인 필요 |
| 계약관리 대시보드/목록/상세 | /pages/contract/* | 계약관리 전체 화면 설명 없음 |
| 발주 리스트/현황 | /pages/order/* | 발주관리 전체 화면 설명 없음 |
| 재고 현황 | /:orderNo/stock | 요구사항 미매핑 — 기능 범위 확인 필요 |
