준비되었습니다. 응답해 주신 내용은 시스템 설계에 직접 반영되므로, 평소

부탁드립니다.

본 설문에서 사용되는 주요 용어를 안내드립니다. 설문 작성 시 참고해
주십시오.

  ------------------ ----------------------------------------------------
  **용어**           **설명**

  **현행 WIMS**      현재 사용 중인 기존 창호/커튼월 관리 시스템. 자재
                     등록과 기본적인 견적/발주 기능을 제공합니다.

  **WIMS 2.0**       새롭게 개발 예정인 차세대 시스템. 현행 시스템의
                     불편사항을 개선하고 새로운 기능을 추가하는 것이
                     목표입니다.

  **BOM**            Bill of Materials(자재명세서). 하나의 제품을 만들기
                     위해 필요한 모든 원자재, 부품, 공정을 계층적으로
                     정리한 목록입니다. (현행 시스템에는 없는 기능으로,
                     2.0에서 새로 도입 예정)

  **MES**            Manufacturing Execution System(제조실행시스템).
                     공장의 생산 현장에서 공정, 자재, 품질 등을 실시간으로
                     관리하는 시스템으로, WIMS 2.0과 데이터를 주고받을
                     예정입니다.

  **CAD**            Computer-Aided Design(컴퓨터 설계 프로그램). AutoCAD
                     등 건축 도면을 작성하는 프로그램을 말합니다.

  **GMB**            현재 제조 도면을 출력할 때 사용하는 외부
                     프로그램입니다. 자재 절단 정보 등을 수동으로
                     입력해야 합니다.

  **최적 절단**      원자재(알루미늄 바 등)를 잘라서 제품을 만들 때,
                     버리는 부분(자투리)을 최소화하도록 계산하는 것을
                     말합니다.

**2-2. 견적 설계**

현행 시스템에서도 견적 기능을 사용하고 계십니다. WIMS 2.0에서는 CAD
도면(건축 설계도)을 웹에서 바로 보면서, 도면 위의 창호 위치에 제품을
지정하면 필요한 자재와 비용이 자동 계산되는 것을 목표로 합니다.

+-----------------------------------------------------------------------+
| **Q7.** 현재 견적 작업은 어떤 순서로 진행하고 계십니까? 각 단계에서   |

+-----------------------------------------------------------------------+
| *작성 예시) \"도면 수신 → 도면에서 창호 위치/사양 확인 → 자재 수량    |
| 산출(Excel) → 단가 적용 → 견적서 작성\"*                              |
|                                                                       |
| *작성 예시) \"도면에서 창호 사양을 파악하는 데 가장 시간이 많이       |

|                                                                       |
| *작성 예시) \"견적 금액을 계산할 때 자재비 + 가공비 + 설치비를 각각   |

**2-3. 발주 관리**

현행 시스템에서도 발주 기능을 사용하고 계십니다. WIMS 2.0에서는 자재를
자를 때 버려지는 부분(자투리)을 최소화하도록 컴퓨터가 자동 계산해 주고,
그 결과로 발주서가 자동 생성되는 것을 목표로 합니다.
