## 1. As-Is 현행 업무 프로세스

### 1.1 개요

현행 WIMS에는 제조관리 기능이 **전면 미구현** 상태이다. 제작도 산출, 절단 지시서, 공정별 추적, 추가 발주 등의 업무가 WIMS 외부(Excel, 수기 문서, 구두 전달)에서 이루어지고 있다. 설문조사에서 현장 실측/제조 관리 익숙도가 1.33점(매우 미숙)으로 최저치를 기록했다.

### 1.2 현행 업무 흐름도

```mermaid
flowchart TD
    Start([제조관리 시작])

    Start --> A1[발주 확정 정보 수신<br/>⚠️ WIMS 외부 - 이메일/구두]
    A1 --> A2[제작도 수동 산출<br/>⚠️ Excel/CAD 별도 작업]
    A2 --> A3[절단 지시서 수동 작성<br/>⚠️ 종이 문서/Excel]

    A3 --> B1[공장 전달<br/>⚠️ 인쇄물/이메일]
    B1 --> B2[공정별 진행 현황<br/>⚠️ 확인 불가 - 전화/방문]

    B2 --> C1{추가 발주<br/>필요?}
    C1 -->|Yes| C2[추가 발주 수동 처리<br/>⚠️ 발주관리와 별도 진행]
    C1 -->|No| C3[제조 완료 확인<br/>⚠️ 구두 보고]
    C2 --> C3

    C3 --> End([종료])
```

## 2. To-Be 목표 업무 프로세스

### 2.1 개요

WIMS 2.0에서 제조관리를 전면 신규 구현한다. 현장실측 데이터를 REST API로 자동 수신하여 제작도 산출, 절단 지시서 생성, 추가 발주 연계까지의 전 과정을 시스템화한다. MES 시스템과의 REST API 연동으로 실시간 생산 데이터를 교환한다.

### 2.2 목표 업무 흐름도

```mermaid
flowchart TD
    Start([제조관리 시작])

    Start --> A1[현장실측 데이터 수신<br/>REST API 자동 수신 FR-MF-001]
    A1 --> A2[수신 데이터 검증 + 이상값 알림<br/>FR-MF-002]
    A2 --> A3{데이터<br/>유효?}
    A3 -->|이상값| A4[재측정 요청 발송]
    A3 -->|정상| A5[실측 데이터 확정]
    A4 --> A1

    A5 --> B1[정형 제품 제작도 자동 출력<br/>FR-MF-003]
    B1 --> B2{비정형<br/>제품?}
    B2 -->|Yes| B3[비정형 제품 수동 편집기<br/>FR-MF-004]
    B2 -->|No| B4[제작도 확정]
    B3 --> B4

    B4 --> C1[MES 연동 제작도 데이터 반영<br/>REST API FR-MF-005]
    C1 --> C2[실측 기반 최적 절단 조합 재계산<br/>FR-MF-006]
    C2 --> C3[최종 절단 데이터 산출<br/>FR-MF-007]
    C3 --> C4[제조 현장용 절단 지시서 출력<br/>FR-MF-008]

    C4 --> D1[1차 발주량 vs 최종 발주량 자동 비교<br/>FR-MF-009]
    D1 --> D2{추가 발주<br/>필요?}
    D2 -->|Yes| D3[추가 발주서 출력 및 처리<br/>발주관리 연계 FR-MF-010]
    D2 -->|No| D4[제조 완료 처리]
    D3 --> D4

    D4 --> End([제조관리 완료])
```
