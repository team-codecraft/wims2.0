# WIMS 현행 시스템 사이트맵 및 페이지별 기능 분석

> **분석 대상**: unique-fe-static (Vue.js 프론트엔드)
> **분석일**: 2026-03-26
> **소스 기준**: master 브랜치 (최종 커밋 2023-08-07)
> **기술 스택**: Vue.js (Options API) + Vuex + Vue Router + Express.js(백엔드) + MariaDB
> **검증**: GNB/LNB 메뉴 정의(store/modules/ui.js) 및 라우터 교차 검증 완료

### 페이지 상태 범례

| 표시 | 의미 | 설명 |
|:---:|------|------|
| ✅ | ACTIVE | GNB/LNB 메뉴에서 직접 접근 가능한 페이지 |
| 🔗 | INTERNAL | 메뉴에 없지만, 다른 페이지에서 프로그래밍적으로 이동하는 페이지 |
| ⚠️ | HIDDEN | 라우트 존재하나 메뉴에서 주석처리(비활성화)된 페이지 |
| 🛠️ | DEV-ONLY | 개발/테스트 전용 페이지 (운영 미사용) |
| ❌ | ORPHANED | 라우트 없이 메뉴 정의만 남은 유령 메뉴 |

## 0. 검증 결과 요약

### GNB 메뉴 정의 (`store/modules/ui.js`)

**활성 메뉴 7개:**
1. 프로젝트 (`/pages/project`) — pageId: 10000
2. 자재관리 (`/pages/material`) — pageId: 20000
3. 제품관리 (`/pages/product`) — pageId: 30000
4. 계약관리 (`/pages/contract`) — pageId: 70000
5. 발주관리 (`/pages/order`) — pageId: 80000
6. 거래처 (`/pages/partners`) — pageId: 40000
7. 관리 (`/pages/system`) — pageId: 60000

**비활성(주석처리) 메뉴 1개:**
- ~~수요기관 (`/pages/issuer`)~~ — pageId: 50000 → ❌ 라우트 자체 미존재

### 서브탭 비활성(주석처리) 항목

| 모듈 | 비활성 탭 | 경로 | 라우트 존재 |
|------|-----------|------|:-----------:|
| 자재관리 | 부자재 | `/pages/material/sub` | ✅ 존재 |
| 제품관리 | 인증만료 제품 (단종) | `/pages/product/discontinued` | ✅ 존재 |
| 거래처 | 부자재 거래처 | `/pages/partners/sub` | ✅ 존재 |

→ 이 3개 탭은 라우트는 정상 존재하지만 메뉴에서 의도적으로 숨김 처리됨 (기능 보류 또는 미사용 상태)

### 개발 전용 페이지

| 경로 | 컴포넌트 | 용도 |
|------|----------|------|

→ GNB/LNB에서 링크 없음, 어디에서도 참조되지 않음. 개발 시 직접 URL 입력으로만 접근

## 2. 페이지별 상세 기능 분석

### 2.1 인증 (Authentication)

| 페이지 | 경로 | 컴포넌트 | 기능 |
|--------|------|----------|------|
| 로그인 | `/signIn` | `SignIn.vue` → `SignInForm.vue` | 이메일/비밀번호 로그인, 세션 기반 인증 |
| 회원가입 | `/join` | `join.vue` | 초대 코드 기반 회원가입, 이메일 발송 |

**API**: `POST /auth` (로그인), `GET /auth` (현재 사용자), `PUT /auth` (프로필 수정), `PUT /auth/password` (비밀번호 변경), `GET/POST /auth/code/:code` (초대 코드 검증/가입)

### 2.2 프로젝트관리 (Project Management)

| 페이지 | 경로 | 컴포넌트 | 기능 |
|--------|------|----------|------|
| 프로젝트 목록 | `/pages/project` | `ProjectList.vue` | 프로젝트 생성/검색/필터/삭제, 유형별(유치원~군부대) 필터, 상태별 필터, 탭 분류(전체/내/완료/즐겨찾기) |
| 기본정보 | `/pages/project/:projectNo` | `CommonInfo.vue` | 프로젝트 기본정보(유형/이름/주소/일정), 고객정보, 계약유형(조달/사급/물품입찰/공사입찰/수의), 자재등급(우수/마스일반), 담당자 배정, 자동저장 |
| 체크리스트 | `/pages/project/:projectNo/checklist` | `CheckList.vue` | 공통/설계노트 2개 탭, 버전별 필터, 이미지+내용+일자+담당자, 확인자 추적, 도면 마커 연동, CRUD+다운로드 |
| 파일관리 | `/pages/project/:projectNo/files` | `FileInfo.vue` | 폴더/파일 트리 구조, 폴더 생성/이름변경, 파일 업로드/다운로드/삭제, 유형/크기/수정일/업로더 표시 |
| 견적 | `/pages/project/:projectNo/estimate` | `EstimateInfo.vue` | 프로젝트에 연결된 견적 목록, 견적 상세 이동 |
| 발주 | `/pages/project/:projectNo/orderForm` | `OrderForm.vue` | 프로젝트 발주서 관리 |
| 실행예산 | `/pages/project/:projectNo/calculate` | `CalculateForm.vue` | 실행예산 문서 일괄 다운로드, 예산 계산 관리 |

**API**: `GET/POST /project`, `GET/PUT /project/:projectNo`, `GET /project/:projectNo/common`, `POST/DELETE /project/:projectNo/members`, `POST/PUT /project/:projectNo/checklist`, `POST /project/:projectNo/directory`, `POST /project/:projectNo/status`
